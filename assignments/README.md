This folder contains all course assignments organized by module.

## Assignment Types

- **Tutorial (T)**: Guided learning exercises
- **Lab (LAB)**: Hands-on practice with instructor support
- **Homework (HW)**: Independent practice assignments
- **Project**: Major assessments

## Naming Convention

Files follow the pattern: `M{module}{type}{number}_{name}.md`

Examples:
- `M1T1_HelloWorld.md` - Module 1, Tutorial 1, Hello World
- `M2LAB1_CrateManufacturing.md` - Module 2, Lab 1, Crate Manufacturing
