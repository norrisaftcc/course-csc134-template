# Module 0: Introduction & Environment Setup

**Week:** 1
**Status:** Complete

## Overview

This module introduces students to the course environment and essential Git workflows. No prior programming experience required.

## Deliverables

| File | Type | Description |
|------|------|-------------|
| `01_workspace_setup.md` | Tutorial | GitHub Codespaces setup, customization, first compile |
| `02_first_pull_request.md` | Lab | Complete GitHub workflow: fork, branch, commit, PR |

## Learning Objectives

By completing this module, students will:
- Set up and customize a cloud-based development environment
- Compile and run a simple C++ program
- Understand the fork-branch-commit-PR workflow
- Make their first contribution to a shared repository

## Pacing

- `01_workspace_setup.md`: 1-2 hours (includes setup time)
- `02_first_pull_request.md`: 1 hour

## Prerequisites

- GitHub account
- Web browser (Chrome, Firefox, Safari, or Edge)
- No prior programming experience required
