# m4 faq
Common Issues and Solutions
Issue 1: Menu Doesn't Loop
Problem: Menu shows once then program ends
Solution: Wrap menu in do-while or while loop
Issue 2: Can't Exit Loops
Problem: No way to return to main menu
Solution: Add "back" or "return" option in submenus
Issue 3: Statistics Don't Update
Problem: Counters not incrementing
Solution: Check scope - are variables resetting?
Issue 4: Input Buffer Issues
Problem: Program skips input or reads old input
Solution: Use cin.ignore() after reading numbers before strings

Final Advice
The Three Laws of Loops

Every loop must have an exit - No infinite loops!
Validate everything - Users will enter garbage
Test the boundaries - First iteration, last iteration, and beyond

Start Simple, Build Up

Get a basic menu working
Add one feature at a time
Test after each addition
