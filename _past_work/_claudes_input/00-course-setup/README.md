This course embraces:
- **Learning through building** - Every concept connects to a real project
- **Celebrating mistakes** - Bugs are teachers in disguise
- **Progressive challenge** - Start simple, build toward complexity
- **Inclusive support** - Multiple learning styles and backgrounds welcome

Throughout this course, you'll build an evolving Role-Playing Game (RPG) that naturally introduces programming concepts:

- **Weeks 1-2**: Character stats system (variables, I/O)

## Course Structure

### Phase 1: Foundations (Weeks 1-8) - "Earn Your C"
- Build core programming skills
- **NO AI code generation** - understanding comes first
- Focus on syntax, logic, and problem-solving

### Phase 2: AI-Assisted Engineering (Weeks 9-12) - "Build Your B"
- Introduction to AI tools as programming assistants
- Learn to generate, understand, and modify code
- Emphasis on STL algorithms and modern C++

### Phase 3: Advanced Concepts (Weeks 13-15) - "Achieve Your A"
- Object-oriented design principles
- SOLID principles in game development
- Professional software engineering practices

### Phase 4: Demonstration (Week 16) - "Badge Defense"
- Portfolio presentation
- Code review and explanation
- Capstone project defense

## Success Strategies

1. **Code every day** - Even 15 minutes maintains momentum
2. **Debug with purpose** - Every error teaches something
3. **Ask questions** - Confusion is the first step to understanding
4. **Share struggles** - You're not alone in finding this challenging
5. **Celebrate small wins** - Compiling without errors IS an achievement!
